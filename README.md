# Rosheed Fagbenro - 3D Developer Portfolio 🚀

Welcome to the open-source repository of my interactive, 3D developer portfolio! This project showcases a blend of modern web technologies, pushing the boundaries of what's possible in the browser with 3D graphics, physics-based interactions, and smooth animations.

![Portfolio-Preview](https://github.com/user-attachments/assets/3c4557e7-6392-4928-b8a9-7b2476ef4edd)

## ✨ Features

- **Immersive 3D Experience**: Built with Three.js and React Three Fiber.
- **Physics Integration**: Realistic object interactions powered by Rapier and Cannon physics engines.
- **Advanced Animations**: Complex, timeline-based animations driven by GSAP.
- **Smooth Scrolling**: Silky-smooth custom scrolling experience using Lenis.
- **Post-Processing**: Enhanced visual effects and WebGL rendering.
- **Fully Responsive**: Optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack

- **Framework**: React 18 (TypeScript) + Vite
- **3D & WebGL**: Three.js, React Three Fiber (`@react-three/fiber`), React Three Drei (`@react-three/drei`)
- **Physics**: Rapier (`@react-three/rapier`), Cannon (`@react-three/cannon`)
- **Animation**: GSAP (GreenSock Animation Platform)
- **Scrolling**: Lenis
- **Styling**: Vanilla CSS / HTML
- **Post-Processing**: React Three Postprocessing

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm (or yarn/pnpm) installed on your machine.
- Node.js (v18 or higher recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rosheed124/Curriculum-Vitae.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd "Rosheed Fagbenro Portfolio"
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   Navigate to `http://localhost:5173` to view the portfolio.

## ⚠️ Important Note regarding GSAP

I have modified the GSAP club plugins with trial plugins for development purposes. Please note that **with the trial plugin, you cannot host or deploy the site** 🔴. 

If you wish to deploy your own version or use the premium features in production, you will need to acquire the official GSAP Club plugins. Check out the installation guide here: [https://gsap.com/docs/v3/Installation/](https://gsap.com/docs/v3/Installation/)
